<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/akb3y/Thyme.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Use `.env.example` to create your .env file
   ```js
   PORT =
   DB_NAME = 
   ```
5. Start the app.
   ```js
   npm start
   ```
